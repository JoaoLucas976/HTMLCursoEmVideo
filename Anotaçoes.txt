HTML é uma linguagem focada em conteúdo. Texto, imagem, listas e etc
CSS é uma linguagem focada em design. Cores, sombras, tamanho, posicionamento e etc
Javascript é uma linguagem focada em interatividade. menus, animações, popups, validações e etc

Cada lingua de código em uma classe do CSS é uma declaração. Declarações são propriedades e valores

FRONT-END, BACK-END, FULLSTACK

PHP, NodeJS, C#, Python, Ruby, Java -  Server Side (Back-end){O servidor é quem interpreta o site, o site é mais inteligente, pois usa dados do servidor pra montar a página, página dinâmica}
